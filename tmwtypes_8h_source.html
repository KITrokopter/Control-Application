00106 /* When used with Real Time Workshop generated code, this
00107  * header file can be used with a variety of compilers.
00108  *
00109  * The compiler could be for an 8 bit embedded processor that
00110  * only had 8 bits per integer and 16 bits per long.
00111  * In that example, a 32 bit integer size is not even available.
00112  * This header file should be robust to that.   
00113  *
00114  * For the case of an 8 bit processor, the preprocessor
00115  * may be limited to 16 bit math like its target.  That limitation 
00116  * would mean that 32 bit comparisons can't be done accurately.  
00117  * To increase robustness to this, comparisons are done against
00118  * smaller values first.  An inaccurate 32 bit comparison isn't
00119  * attempted if the 16 bit comparison has already succeeded.
00120  *
00121  * Limitations on preprocessor math can also be stricter than
00122  * for the target.  There are known cases where a compiler
00123  * targeting processors with 64 bit longs can't do accurate
00124  * preprocessor comparisons on more than 32 bits.  
00125  */
